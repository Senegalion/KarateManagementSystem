# KarateManagementSystem

KarateManagementSystem is a web application that combines a Java Spring backend and an Angular frontend to manage a karate club system.

## Project Structure

- **KarateManagementSystem (Backend)**
  - This folder contains all the Java and Spring code for the backend.

- **KarateManagementSystemAngular (Frontend)**
  - This folder contains all the Angular code for the frontend.

## Technologies Used

- **Backend**
  - Java
  - Spring Boot
  - Hibernate
  - RESTful APIs

- **Frontend**
  - Angular
  - HTML, CSS, TypeScript

## Setup and Installation

### Prerequisites

- Java JDK 11 or higher
- Node.js and npm
- Angular CLI
- Spring Boot

### Backend Setup

1. Navigate to the backend directory:
    ```sh
    cd KarateManagementSystem
    ```
2. Build and run the backend:
    ```sh
    ./mvnw spring-boot:run
    ```

### Frontend Setup

1. Navigate to the frontend directory:
    ```sh
    cd KarateManagementSystemAngular
    ```
2. Install the dependencies:
    ```sh
    npm install
    ```
3. Run the Angular development server:
    ```sh
    ng serve
    ```
4. Open your browser and navigate to `http://localhost:4200`.

## Usage

1. Start the backend server.
2. Start the frontend development server.
3. Open your browser and navigate to `http://localhost:4200` to access the application.
